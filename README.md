# CheekyTarot

A simple WPF desktop tarot app built in C#.

## Features
- Random tarot card draw
- Upright and reversed meanings
- Data loaded from JSON

## Purpose
Created as a personal project while refreshing core C# and WPF skills.

## Future Improvements
- Card images
- Personality modes
- Deck system (no repeats)

### Disclaimer
This app is meant for entertainment purposes only and should not be taken as life advice
